	`libgmp.so.3.5.2`
	`libgmp.so.3 -> libgmp.so.3.5.2`
	`libgmp.so -> libgmp.so.3.5.2`

Możliwe wymagania
- biblioteka `libgmp` (`libgmp.so`)
- liblioteka `libgmp` w wersji 3 (`libgmp.so.3`)
- implementacja może się zmieniać w ramach wersji np. -> `3.5.3` -> `3.6.2` itd.
